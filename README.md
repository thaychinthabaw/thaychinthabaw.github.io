# Thaychinthabaw Website
ဤသည်မှာ "Thaychinthabaw" စာအုပ်၏ ပင်မဝဘ်ဆိုက်ဖြစ်သည်။
- တည်ဆောက်သူ: ဦးပညာဝရ
- နည်းပညာ: HTML, CSS, JavaScript
- ရည်ရွယ်ချက်: ဝိပဿနာ ဆိုင်ရာ အကြောင်းအရာများကို မျှဝေရန်။

(တရားစာများကို လွတ်လပ်စွာ ပြန်လည်မျှဝေခွင့်ပြုသည် လုံးဝ အရေးယူမည် မဟုတ်ပါ) 



ဒီအောက်က လင့်က အခု thaychinthabaw.github.io အတွက်မူရင်းကုဒ် လင့်ပါ

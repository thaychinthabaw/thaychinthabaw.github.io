ကိုယ်ရေးအကျဉ်း

အရှင်ပညာဝရသည် မွေးသက္ကရာဇ် ( ၃၀-၁၂-၁၉၉၉ )ခုနှစ်တွင် အဖဦးအေးသိန်း၊ အမိဒေါ်အေးအေးမာ တို့မှ မွေးဖွားခဲ့သည်။ မွေးဖွားရာဒေသမှာ ရန်ကုန်မြို့၊ သာကေတမြို့နယ် ဖြစ်သည်။ ငယ်နာမည်မှာ မောင်ပိုလေး ဖြစ်ပါသည်။ ကျောင်းနာမည်မှာ မောင်ဖြိုးကျော်ထက် ဖြစ်ပါသည်။

အရှင်ပညာဝရသည် အသက် ( ၂၃ )နှစ်၊ ( ၅.၁၁.၂၀၂၃ )တွင် သာသနာဘောင်သို့ ဝင်ရောက်ကာ အများအကျိုးပြုလုပ်ငန်းများကို ဆောင်ရွက်လျက်ရှိသည်။ သေခြင်းသဘော ဝိပဿနာအခြေခံကို မျှဝေခြင်း၊ အကြံပေးခြင်းနှင့် ဗဟုသုတဖြန့်ဝေမှုများကို ဆောင်ရွက်လျက်ရှိသည်။

"https://thaychinthabaw.github.io" ဝက်ဆိုဒ်ဖြင့် "သေခြင်းသဘော ဝိပဿနာအခြေခံ" စာမူကို လွယ်လွယ်ကူကူ ဖတ်နိုင်အောင် လုပ်ဆောင်လျက်ရှိသည်။

လူမှုမီဒီယာပေါ်တွင် “သေခြင်းသဘော | ဝိပဿနာအခြေခံ ☸️” အမည်ဖြင့် အခမဲ့ မျှဝေပေးလျက်ရှိသည်။ ယင်းအဖွဲ့အစည်းသည် အကျိုးအမြတ်မလိုသည့် အများအကျိုးပြုသော ပရဟိတရည်ရွယ်ချက်ဖြင့် တည်ထောင်ထားခြင်း ဖြစ်သည်။ လက်ရှိတွင် YouTube, TikTok, Telegram နှင့် Facebook တို့တွင် ပညာဖြန့်ဝေမှုများကို ဆက်လက်လုပ်ဆောင်လျက်ရှိသည်။

အဓိကရည်ရွယ်ချက်

ဝိပဿနာ အကြောင်းကို သိလိုသူများအတွက် လွယ်ကူစွာ လေ့လာနိုင်အောင် လုပ်ဆောင်လျက်ရှိပါတယ်။

လေ့လာမှုကဏ္ဍများ

  • • Tipitaka Myanmar Application မှ သုတ္တန် နိကာယ် (၅)ကျမ်း မြန်မာပြန် များကို (၁)ကြိမ် ဖတ်ခဲ့ပါတယ်။
  • • Tipitaka Myanmar Application မှ ပဋိသမ္ဘိဒါမဂ် မြန်မာပြန်ကို အထူးပြု လေ့လာခဲ့ပါတယ်။
  • • နိကာယ် (၅)ကျမ်းမှ နည်းစနစ်များကို လိုက်နာ၍ ရိပ်သာတွင် (၁)နှစ် တရားအားထုတ်ခဲ့ပါတယ် တရားအားထုတ်ခဲ့ပါတယ်။။
  • • တရားအားထုတ်စဉ်က သိခဲ့ရသော ဗာဟိယသုတ်အရ တရားရှုမှတ်နည်းနှင့် အပရိဟာနိယသုတ်အရ နိုးကြားခြင်း၌ ယှဉ် တဲ့ အားထုတ်နည်း တို့ကို မျှဝေပေးလျက် ရှိပါတယ်။
